## Overview

The **Circle Crop Image** tool is a free, browser-based **circular image cropper** that allows you to transform any image into a perfect circle with a transparent background. All processing happens locally in your browser, ensuring privacy and speed.

## Getting Started

### Accessing the Tool

You can access the Circle Crop Image tool in two ways:

1. Visit the [Circle Crop page](/circle-crop) directly
2. Use the tool directly on the [homepage](/)

### Supported File Formats

The tool accepts the following image formats:

- **JPG/JPEG** - Standard photo format
- **PNG** - Images with transparency support
- **WEBP** - Modern web format
- **GIF** - Animated or static images

<Callout title="Output Format">
All images are exported as PNG format with transparent background, regardless of the input format.
</Callout>

## How to Use

### Step 1: Upload Your Image

1. Click the "Upload image" button
2. Select an image file from your device
3. The image will load automatically in the preview area

### Step 2: Adjust Zoom

Control the zoom level using:

- **Mouse Wheel**: Scroll up to zoom in, scroll down to zoom out
- **Zoom Slider**: Use the slider control for precise adjustments

**Zoom Range**: 0.5x to 2.0x

### Step 3: Position the Image

- Click and drag the image within the preview area
- The circular border shows the exact crop area
- Position your subject where you want it in the final circle

### Step 4: Download

1. Click the "Download Circle Image" button
2. Your circular image will download as a PNG file
3. The file will have a transparent background

## Tips for Best Results

### Image Selection

- Start with high-resolution images (minimum 400x400 pixels)
- Choose images with clear, centered subjects
- Avoid overly cluttered images
- Ensure good contrast between subject and background

### Composition

- Center your main subject
- Use zoom to focus on important elements
- Leave some breathing room around edges
- Check the preview before downloading

### Technical Tips

- Use PNG source images when possible for best quality
- For portraits, position faces in the upper third
- For products, center the main item
- Adjust zoom gradually for precise control

## Keyboard Shortcuts

While using the tool:

- **Mouse Wheel**: Zoom in/out

- **Scroll**: Prevented on preview area (zooms instead)

## Troubleshooting

### Image Won't Load

- Check that the file is a supported format
- Ensure the file isn't corrupted
- Try a different image file

### Zoom Not Working

- Make sure your mouse is over the preview area
- Check that JavaScript is enabled
- Try refreshing the page

### Download Issues

- Check your browser's download settings
- Ensure pop-up blockers aren't interfering
- Try a different browser if problems persist
