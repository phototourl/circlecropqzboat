## 概述

**圆形图片裁剪工具**是一个免费的、基于浏览器的**圆形图片裁剪器**，允许您将任何图片转换为具有透明背景的完美圆形。所有处理都在您的浏览器本地完成，确保隐私和速度。

## 开始使用

### 访问工具

您可以通过两种方式访问圆形图片裁剪工具：

1. 直接访问[圆形裁剪页面](/circle-crop)
2. 在[主页](/)上直接使用该工具

### 支持的文件格式

该工具接受以下图片格式：

- **JPG/JPEG** - 标准照片格式
- **PNG** - 支持透明度的图片
- **WEBP** - 现代网页格式
- **GIF** - 动画或静态图片

<Callout title="输出格式">
所有图片均以透明背景的 PNG 格式导出，无论输入格式如何。
</Callout>

## 使用方法

### 步骤 1：上传您的图片

1. 点击"上传图片"按钮
2. 从您的设备中选择图片文件
3. 图片将自动加载到预览区域

### 步骤 2：调整缩放

使用以下方式控制缩放级别：

- **鼠标滚轮**：向上滚动放大，向下滚动缩小
- **缩放滑块**：使用滑块控件进行精确调整

**缩放范围**：0.5x 到 2.0x

### 步骤 3：定位图片

- 在预览区域内点击并拖动图片
- 圆形边框显示确切的裁剪区域
- 将您的主题定位在最终圆圈中您想要的位置

### 步骤 4：下载

1. 点击"下载圆形图片"按钮
2. 您的圆形图片将作为 PNG 文件下载
3. 文件将具有透明背景

## 功能

### 隐私优先处理

所有图片处理都在**您的浏览器本地**完成。这意味着：

- 您的图片永远不会离开您的设备
- 不上传到外部服务器
- 完全的隐私和安全
- 没有文件大小限制

### 高质量输出

该工具在处理过程中保持图片质量：

- 高分辨率导出（最高 4096x4096 像素）
- 平滑的图片渲染
- 保留原始图片细节
- 专业级结果

### 透明背景

所有导出的图片都具有：

- 带 alpha 通道的 PNG 格式
- 透明背景（无白色边框）
- 适合在任何背景颜色上使用
- 适合设计项目

### 无水印

- 完全免费使用
- 无需注册
- 不添加水印
- 无限下载

## 最佳效果提示

### 图片选择

- 从高分辨率图片开始（最小 400x400 像素）
- 选择具有清晰、居中主题的图片
- 避免过于杂乱的图片
- 确保主题和背景之间有良好的对比度

### 构图

- 居中您的主要主题
- 使用缩放聚焦重要元素
- 在边缘留一些呼吸空间
- 下载前检查预览

### 技术技巧

- 尽可能使用 PNG 源图片以获得最佳质量
- 对于肖像，将脸部定位在上三分之一
- 对于产品，居中主要物品
- 逐渐调整缩放以进行精确控制

## 浏览器兼容性

圆形图片裁剪工具适用于所有现代浏览器：

- Chrome（推荐）
- Firefox
- Safari
- Edge
- Opera

<Callout title="浏览器要求">
必须启用 JavaScript。该工具使用 HTML5 Canvas API 进行图片处理。
</Callout>

## 键盘快捷键

使用工具时：

- **鼠标滚轮**：放大/缩小
- **点击 + 拖动**：重新定位图片
- **滚动**：在预览区域被阻止（改为缩放）

## 故障排除

### 图片无法加载

- 检查文件是否为支持的格式
- 确保文件未损坏
- 尝试不同的图片文件

### 缩放不工作

- 确保鼠标在预览区域上方
- 检查是否启用了 JavaScript
- 尝试刷新页面

### 下载问题

- 检查浏览器的下载设置
- 确保弹出窗口阻止程序没有干扰
- 如果问题持续，尝试不同的浏览器
