## 一般问题

## 文件格式问题

<Accordions>
  <Accordion id="supported-formats-zh" title="支持哪些图片格式？">
    该工具接受 JPG、PNG、WEBP 和 GIF 格式。您可以上传这些格式中的任何一种。
  </Accordion>
  
  <Accordion id="output-format-zh" title="下载的图片是什么格式？">
    所有图片均以透明背景的 PNG 文件下载，无论您上传的原始格式如何。
  </Accordion>
  
  <Accordion id="why-png-zh" title="为什么是 PNG 格式？">
    PNG 格式支持透明度，这对于圆形图片至关重要。这允许您在任何背景颜色上使用圆形图片，而不会有白色边框或不需要的背景。
  </Accordion>
  
  <Accordion id="file-size-zh" title="有文件大小限制吗？">
    没有，由于所有处理都在您的浏览器中完成，因此没有文件大小限制。但是，非常大的图片可能需要更长的处理时间，具体取决于您设备的性能。
  </Accordion>
</Accordions>

## 使用问题

<Accordions>
  <Accordion id="how-to-use-zh" title="如何使用该工具？">
    1. 点击"上传图片"并选择您的图片
    2. 使用鼠标滚轮或缩放滑块调整缩放

    4. 点击"下载圆形图片"以保存您的圆形图片
  </Accordion>
  
  <Accordion id="zoom-range-zh" title="缩放范围是多少？">
    您可以从 0.5x（缩小）缩放到 2.0x（放大），让您完全控制圆形裁剪中显示的内容。

  <Accordion id="positioning-zh" title="我可以移动图片吗？">
    可以，只需在预览区域内点击并拖动图片，即可将其精确定位到最终圆圈中您想要的位置。

  <Accordion id="preview-zh" title="预览显示什么？">
    预览显示图片的方形视图，带有白色圆形边框，指示确切的裁剪区域。圆圈内的所有内容都将包含在您的最终图片中。

</Accordions>

## 质量和分辨率

<Accordions>
  <Accordion id="image-quality-zh" title="图片质量会保持吗？">
    是的，该工具在处理过程中保持高图片质量。它使用高质量渲染，可以导出最高 4096x4096 像素的图片，同时保留细节。
  </Accordion>
  
  <Accordion id="resolution-zh" title="我应该使用什么分辨率？">
    为了获得最佳效果，请使用高分辨率源图片：
    - 头像：最小 400x400 像素

    像素化通常发生在源图片分辨率太低时。始终从高质量的源图片开始以获得最佳效果。
  </Accordion>
</Accordions>

## 故障排除

<Accordions>
  <Accordion id="image-wont-load-zh" title="我的图片无法加载">
    - 检查文件是否为支持的格式（JPG、PNG、WEBP、GIF）
    - 确保文件未损坏
    - 尝试不同的图片文件
    - 刷新页面并重试
  </Accordion>
  
  <Accordion id="zoom-not-working-zh" title="缩放不工作">
    - 确保鼠标在预览区域上方
    - 检查浏览器中是否启用了 JavaScript
    - 尝试使用缩放滑块而不是鼠标滚轮
    - 如果问题持续，刷新页面
  </Accordion>
  
  <Accordion id="poor-quality-zh" title="输出质量很差">
    - 从更高分辨率的源图片开始
    - 避免过度缩放（保持在合理的缩放级别内）
    - 尽可能使用 PNG 源图片以获得最佳质量
  </Accordion>
</Accordions>
