## General Questions

<Accordions>
  <Accordion id="what-is" title="What is Circle Crop Image?">
    Circle Crop Image is a free online tool that allows you to crop any image into a perfect circle with a transparent background. It's a browser-based **circular image cropper** that processes images locally on your device.
  </Accordion>
  
  <Accordion id="how-it-works" title="How does it work?">
    The tool uses HTML5 Canvas API to process images directly in your browser. You upload an image, adjust the zoom and position, and download a circular PNG image with a transparent background. All processing happens locally - your images never leave your device.
  </Accordion>
  
  <Accordion id="is-it-free" title="Is it really free?">
    Yes, completely free! No signup required, no watermarks, no hidden fees, and no limitations on how many images you can process.
  </Accordion>
  
  <Accordion id="privacy" title="Are my images private?">
    Absolutely. All image processing happens locally in your browser. Your images are never uploaded to any server, ensuring complete privacy and security.
  </Accordion>
</Accordions>

## File Format Questions

<Accordions>
  <Accordion id="supported-formats" title="What image formats are supported?">
    The tool accepts JPG, PNG, WEBP, and GIF formats. You can upload images in any of these formats.
  </Accordion>
  
  <Accordion id="output-format" title="What format are downloaded images?">
    All images are downloaded as PNG files with transparent backgrounds, regardless of the original format you uploaded.
  </Accordion>
  
  <Accordion id="why-png" title="Why PNG format?">
    PNG format supports transparency, which is essential for circular images. This allows you to use your circular images on any background color without white borders or unwanted backgrounds.
  </Accordion>
  
  <Accordion id="file-size" title="Is there a file size limit?">
    No, there's no file size limit since all processing happens in your browser. However, very large images may take longer to process depending on your device's capabilities.
  </Accordion>
</Accordions>

## Usage Questions

<Accordions>
  <Accordion id="how-to-use" title="How do I use the tool?">
    1. Click "Upload image" and select your image
    2. Use mouse wheel or zoom slider to adjust zoom
    3. Click and drag to position the image
    4. Click "Download Circle Image" to save your circular image
  </Accordion>
  
  <Accordion id="zoom-range" title="What's the zoom range?">
    You can zoom from 0.5x (zoom out) to 2.0x (zoom in), giving you full control over what appears in your circular crop.
  </Accordion>
  
  <Accordion id="positioning" title="Can I move the image?">
    Yes, simply click and drag the image within the preview area to position it exactly where you want it in the final circle.
  </Accordion>
  
  <Accordion id="preview" title="What does the preview show?">
    The preview shows a square view of your image with a white circular border indicating the exact crop area. Everything inside the circle will be included in your final image.
  </Accordion>
</Accordions>

## Quality and Resolution

<Accordions>
  <Accordion id="image-quality" title="Will the image quality be maintained?">
    Yes, the tool maintains high image quality during processing. It uses high-quality rendering and can export images up to 4096x4096 pixels while preserving details.
  </Accordion>
  
  <Accordion id="resolution" title="What resolution should I use?">
    For best results, use high-resolution source images:
    - Profile pictures: Minimum 400x400 pixels
    - Social media: 1000x1000 pixels or higher
    - Print use: 2000x2000 pixels or higher
  </Accordion>
  
  <Accordion id="pixelated" title="Why is my image pixelated?">
    Pixelation usually occurs when the source image is too low resolution. Always start with high-quality source images for the best results.
  </Accordion>
</Accordions>

## Troubleshooting

<Accordions>
  <Accordion id="image-wont-load" title="My image won't load">
    - Check that the file is a supported format (JPG, PNG, WEBP, GIF)
    - Ensure the file isn't corrupted
    - Try a different image file
    - Refresh the page and try again
  </Accordion>
  
  <Accordion id="zoom-not-working" title="Zoom isn't working">
    - Make sure your mouse is over the preview area
    - Check that JavaScript is enabled in your browser
    - Try using the zoom slider instead of mouse wheel
    - Refresh the page if issues persist
  </Accordion>
  
  <Accordion id="poor-quality" title="The output quality is poor">
    - Start with a higher resolution source image
    - Avoid over-zooming (stay within reasonable zoom levels)
    - Use PNG source images when possible for best quality
  </Accordion>
</Accordions>
